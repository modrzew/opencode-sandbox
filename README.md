Run a coding agent (`opencode` or `pi`) in a sandbox powered by [Apple containers](https://github.com/apple/container) with access to `git` and `gh`.

Usage: `ocsbx.sh` to start the default agent, `ocsbx.sh pi` / `ocsbx.sh opencode` to pick one, `ocsbx.sh <hash>` to resume an existing container (its agent is remembered). Set a default `AGENT` in `sandbox.env`.

## 🎯 Usage

### Starting a Fresh Sandbox Session

```bash

```

This will:

1. Resolve the Git repository root and worktree paths
2. Generate a unique container name based on a random hash
3. Mount the working tree and shared `.git` directory (for worktrees)

./ocsbx.sh <hash>
